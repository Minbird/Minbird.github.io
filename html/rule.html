민버드의 좀비 서바이벌 서버 이용 규칙


규칙 위반시 해당 항목에 명시된 횟수만큼의 경고를 받습니다

그렇게 누적된 경고가 3회가 될 시 24시간 밴 처리됩니다

해당 항목에 밴 일수가 적혀있는 경우 경고 없이 적힌 대로 밴 처리됩니다

모든 플레이어의 누적된 경고는 처리된 이후 24시간 후 초기화됩니다



 

욕설 (욕설 기준 보기)

 

 

아래의 경우 경고 2회를 받습니다.
  • 욕설 기준에 해당하는 단어를 채팅에 치거나 음성채팅으로 말 할 경우
  • 욕설 기준에 해당하는 단어의 초성어를 채팅에 치거나 음성채팅으로 말 할 경우
  • 욕설 기준에 해당하지 않지만 욕설 의도가 있다고 판단되는 경우


아래의 경우 최소 30일 밴, 최대 영밴 처리됩니다. 밴 일수는 위 기간 내 관리자의 재량으로 처리됩니다.

  • 위 행위를 관리자에게 하는 경우


아래의 경우 무조건 영밴 처리됩니다.

  • 패드립을 하는 경우


서버 외부에서의 욕설 및 패드립도 신고할 수 있으며 위와 같이 처리됩니다.



 

 

기본 매너

 

 

  • 게임 내 버그를 악용하는 경우, 3일 밴 처리됩니다.
  • 뉴비 배척 행위는 6일 밴 처리됩니다.
  • 탈출 형식의 맵 (ze, zs_obj 등...) 에서 다음 단계로 진행할 수 있음에도 1웨이브(약 3분)동안 움직이지 않고 진행을 방해하는 경우 전원 슬래이된다.
  • rtv, mapvote (맵 변경) 를 부추기거나 강요하는 경우 경고 2회를 받습니다.




 
 

관리자 사칭

 

 

아래의 경우 관리자 사칭으로 판단하고 경고 2회를 받습니다.

  • "경고를 주겠다", "밴 하겠다" 같은 발언을 하는 경우
  • 고의로 닉네임을 관리자와 같게 설정했다고 판단되는 경우


아래의 경우 30일 밴 처리됩니다.

  • 위 항목에 해당되는 행동이 여러번 목격되는 경우



 

어뷰징

 

 

어뷰징은 고의적으로 비정상적인 방법을 통해 포인트나 레벨을 올리는 행위를 말합니다.


  • 맵 구조를 이용해 바리케이드나 아군에게 대미지를 입혀 포인트 및 레벨을 올리는 행위는 경고 1회를 받습니다.



 

Playmusic / 히스 / 음성 채팅

 

 

플레이 뮤직 프로는 채팅에 !pm 을 입력하거나 F9를 눌러 사용할 수 있습니다.


  • 음악에서 나오는 욕설은 허용됩니다.
  • 음성 채팅으로 노래를 부르다가, 중지 요청이 있는 경우 해당 플레이어는 즉시 중지해야 합니다.



 

신고

 

 

신고는 서버에 접속해있는 관리자, 또는 관리자의 스팀 채팅이나 디스코드, 그 외의 연락수단으로 할 수 있습니다.

당시 상황을 증명할 수 있는 스크린샷이나 발생 시각 등을 보내시면 더 빠르게 처리할 수 있습니다.
